These are the source files for some simple things you can do with the streamcontrol.xml file.

They are designed to be able to be used even if you're not able to write ActionScript. You can in theory just move things around into their desired positions on top of a png file (or plain flash shapes as in the examples) to get your desired result. Much more fine tuned (and eye pleasing) results can be accomplished if you have a decent knowledge of Flash.

Scoreboard with score pips supports up to 10 rounds by default.

They embed Arial Unicode by default to support as many languages as possible.
